Installation
============

Python3 and pip are required::

    sudo pip3 install git+https://github.com/grzanka/pycowsay.git


Now you can give it a try::


    run_pycowsay


Uninstallation
============

Python3 and pip are required::

    sudo pip3 uninstall pycowsay
